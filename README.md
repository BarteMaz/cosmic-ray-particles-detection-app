# Aplikacja do detekcji cząstek promieniowania kosmicznego przez wykorzystanie kamer CMOS
### Twórcy 
- Iwo Ropa
- Bartłomiej Mazgaj
- Katsiaryna Naumovich

### Cel Projektu 
Celem naszego projektu jest stworzenie aplikacji na systemy Android i IOS, która poprzez wykorzystanie kamery CMOS znajdującej się w telefonach, umożliwiłaby wykrycie cząsteczek promieniowania kosmicznego.
